# Noro Player

**Noro Player** is a native macOS app: a retro-styled music player for Apple Music that shows your current track, album art, progress, and playback controls.

## Requirements

- **macOS 14** or newer
- **Xcode 15** or newer (to build from source)

## Build

```bash
swift build
```

Or open `Package.swift` in Xcode and run the `NoroPlayer` scheme.

## Download

Head to the [Releases](https://github.com/0oAstro/noro-player-mac/releases) page and grab the latest build.

---

Noro Player originally started as a [Rainmeter](https://www.rainmeter.net/) skin for Windows. Those releases are still up on the [original repo](https://github.com/SunkenInTime/noro-player/releases) if you're looking for them.
